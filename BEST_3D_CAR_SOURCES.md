## Print Quality Guidelines

### Resolution Settings
- **Layer Height:** 0.1-0.2mm for detail
- **Infill:** 20-30% for display models
- **Wall Thickness:** 2-3 perimeters
- **Support:** Tree supports for overhangs

### Material Recommendations
1. **PLA+** - Best for beginners
2. **PETG** - Durable, outdoor safe
3. **ABS** - Strong, vapor smoothable
4. **Resin** - Ultimate detail (SLA/DLP)

### Post-Processing
- Sanding: 220-400 grit progression
- Priming: Automotive filler primer
- Painting: Airbrush for best results
- Clear coat: UV-resistant for display

## Quality Verification Checklist

Before adding to catalog:
- [ ] Manifold geometry (watertight)
- [ ] Proper scale (1:24 or 1:18)
- [ ] Wall thickness >0.8mm
- [ ] No floating parts
- [ ] Print time under 15 hours
- [ ] File size under 100MB
- [ ] Customer reviews >4 stars
- [ ] Test print successful
