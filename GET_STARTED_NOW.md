## Common Issues

**"Still showing a box/generic car"**
- The STL file isn't in the right place
- Check: `public/models/` folder exists
- File name must match exactly

**"Model doesn't load"**
- STL might be too large (>50MB)
- Try a simpler model first
- Check browser console for errors
